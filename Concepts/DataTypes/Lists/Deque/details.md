# Deque


1. `Extends to Queue data structure`
2. `Internally uses Double Linked List`
3. `Advantage: Has O(1) timeComplexity for pop() method`


#### Complexity Analysis

| Operation  | Time Complexity |
|:-----------|:----------------|
| append     | O(1)            |
| appendLeft | O(1)            |
| pop        | O(1)            |
| popLeft    | O(1)            |
| index      | O(N)            |
| indexLeft  | O(N)            |
| insert     | O(N)            |
| remove     | O(N)            |
| count      | O(N)            |
| len(deque) | O(1)            |
